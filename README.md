# data-syntaxes
Valota public data syntaxes

## Fake data
Fake data is created with [json-schema-faker](https://www.npmjs.com/package/json-schema-faker). It uses [faker](https://www.npmjs.com/package/@faker-js/faker), [chance](https://www.npmjs.com/package/chance) and our own [vlt-data-faker](https://github.com/Valota/data-faker-ts)
